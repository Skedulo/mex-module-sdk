# Tutorial

# Create new form
Run `./run.sh create-form {NAME}`

# Install form
Run `./run.sh install-form -f {folderPath}`

# Create new module

Run `./run.sh create-module {NAME}`

# Debugging local
First, you need to get the engine by running: 
`./run.sh get-engine`

Optional: If you have any custom modules, then run
`./run.sh merge-module`

Then run:
`./start-engine.sh`
